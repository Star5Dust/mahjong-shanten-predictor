# 日麻向听数预测模型（Mahjong Shanten Predictor）

基于天凤牌谱训练的对手向听状态预测模型。  
目标：仅通过公开信息（打牌、副露、宝牌等），判断任意非己方玩家当前是否处于 **听牌（0）**、**1向听（1）** 或 **安全状态（2+）**。

## 📊 数据规模
- 来源：Kaggle "tenhou-to-mjai" 数据集（实际为 Majsoul 牌谱）
- 时间：2020–2024 年
- 原始对局：927,039 局（.mjson）
- 生成样本：约 4.6 亿条（每局 ~500 个打牌时刻）

## ⚙️ 特点
- 向听数由 `mahjong.shanten.Shanten` 计算并归为三类：`0`（听牌）、`1`（1向听）、`2+`（安全）
- 输入特征：打牌序列、自摸切标志、副露统计、宝牌指示等（不含真实手牌）
- 样本按完整对局（episode）划分，避免数据泄露
- 模型：基于Transformer的序列分类器
